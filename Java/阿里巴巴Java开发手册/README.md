### 1.5.0华山版

2019.06.19 

1）鉴于本手册是社区开发者集体智慧的结晶，本版本移除阿里巴巴 Java 开发手册的 

限定词“阿里巴巴”。 

2）新增 21 条新规约。比如，switch 的 NPE 问题、浮点数的比较、无泛型限制、锁的 

使用方式、判断表达式、日期格式等。 

3）修改描述 112 处。比如，IFNULL 的判断、集合的 toArray、日志处理等。 

4）完善若干处示例。比如，命名示例、卫语句示例、enum 示例、finally 的 return 

示例等。
